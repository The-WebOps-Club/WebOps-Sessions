What are attributes ??
Attributes are values which give
additional information about the
element
Examples are,
  • href
  • alt
  • name
  • id
  • class
  • style
"id" attribute give the element a unique identity
and "class" gives the element a class literally.
Same class can be given to different elements 
but same id can't be given to different elements
<style>
    .class1 {
        color: red;
    }
    #id1 {
        color: green;
    }
    div {
        color: blue;
    }
</style>
This shows all the elements 
with class, "class1" as red,
element with id, "id1" as green
and "div" element blue
The attribute 'alt' is used to 
display some text instead of

or if an error occured while 
loading

Keyframes... 
Keyframes are like the functions in any other
programming languages.
Syntax:
@keyframes mynewfunc {
0%   /*initial CSS properties*/
11%   /*intermediate CSS propreties*/
39%   /*intermediate CSS properties*/
100%   /*final CSS properties*/
}
For chrome, we need "@-webkit-keyframes"
For Safari and Google Chrome, -webkit-
For Mozilla Firefox, -moz-
For Opera, -o-
These are the layout engine software component
designed to allow web browsers to render web 
pages.
Rendering here means loading the web page.
Note that not all CSS properties are supported 
by all browsers
"animation" property in CSS is used to call our
defined keyframes. It allows us to set the speed 
and number of iterations also
#my_id {
-webkit-animation: mynewfunc 10s infinite;
animation: mynewfunc 10s infinite;
}
